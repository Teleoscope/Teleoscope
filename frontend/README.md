# `frontend` (legacy app)

This directory contains an older frontend codebase kept for historical reference and potential migration work.

## Important

- The active product frontend is `teleoscope.ca/`.
- CI and current development flows target `teleoscope.ca/`, not this directory.
- Do not place new feature work here unless a task explicitly says to modify legacy frontend code.

## If you need to inspect legacy behavior

1. Read the active app first (`teleoscope.ca/README.md`) to avoid duplicating old patterns.
2. Treat this folder as archival/transition code.
3. Open a dedicated migration task if behavior needs to be ported to `teleoscope.ca/`.
